Get Started View on GitHub
Simple by design

Three steps. That's it.

From opening your file to a live website — no terminal, no CMS, no backend required.

1

Open

Open index.html in your browser. Because secrets.js is present, Webby activates automatically and the editor toolbar appears at the top of the page — no installation, no setup.

2

Edit

Click any text to change it, click any image to replace it, or hover between sections and click + Add Section to describe what you want — the AI generates it, styled to match your site.

3

Publish

Hit the Publish button. Webby commits your changes directly to GitHub and your live site updates automatically — usually within about a minute. Every change is version-controlled.

Starting from scratch?

Start a new site
in minutes

Webby doesn't just edit existing sites — it helps you create new ones from scratch. Use the Bootstrap Prompt with Claude.ai to generate a complete, styled index.html from a description of your business.

Attach 2–5 inspiration images — screenshots of sites you like, colour palette images, photos from your work — and Claude will translate them into a cohesive visual design with all Webby attributes pre-wired.

Let's do this

Ready? Here's how
to set it up.

Everything you need is free — GitHub, Claude.ai, and a text editor to create one small file. If you can copy and paste, you can do this.

1

Create a GitHub account and repository

Sign up at github.com if you don't have an account. Create a new public repository for your site. Enable GitHub Pages under Settings → Pages → Deploy from branch → main. Your site will be live at your-username.github.io/your-repo.

2

Generate your index.html with Claude

Open Claude.ai, paste in the Bootstrap Prompt, attach a few inspiration images, and fill in your business details. Claude will return a complete, styled HTML file. Save it as index.html in a folder on your computer.

3

Create your secrets.js file

In the same folder as index.html, create a plain text file called secrets.jsAdd your Gemini API key (free), GitHub personal access token, and repository name. This file stays on your computer only — it is never published.

window.SITE_SECRETS = { geminiKey: "AIza...", githubToken: "ghp_...", repo: "your-username/your-repo", branch: "main" };
4

Open index.html in your browser

Double-click the file. For the best experience use Chrome or Edge — they support auto-saving your edits directly back to disk. The Webby editor will appear automatically at the top of the page.

5

Link your site folder

A banner will appear below the toolbar on first open. Click Select Folder and choose the folder containing your index.html.
Once linked, every edit auto-saves to your local file within a couple of seconds.

6

Hit Publish

When you're happy with your edits, click the Publish button in the Webby toolbar. Webby commits your changes to GitHub and your site will be live within about a minute. Every publish is a git commit with a full history.
